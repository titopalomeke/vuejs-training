Introducción

¿Qué es Vue.js?

Es una librería js para construir interfaces web interactivas.
No es un framework MVC, no tiene controlador, está especializado en la capa de vista y sigue un patrón MVVM.

Data Bindind Reactivo

Vue.js es el bloque central que sincroniza la Vista y el Modelo de manera reactiva, es decir, los cambios en la vista afectan al modelo y viceversa.

¿Cómo hace esto? Mediante directivas insertadas directamente en los tags html.

                    
    <input type="text" id="nombre" v-model="nombre"/>

¿Por qué Vue.js?

  • No es difícil de aprender (aunque mi neurona haya muerto)
  • Su sistema de Binding (Clases, estilos)
  • Es fácil crear componentes
  • Al utilizar un DOM virtual su rendimiento es muy bueno
  • Es posible combinarlo con otras liberías js

Ejemplo

                    
    <div id="welcome-msg">
        <h4>Bienvenido, {{ name }}</h4>

        <input type="text" v-model="name" class="form-control">
    </div>

Creamos una instancia del objeto Vue

                    
    new Vue({
        el: "#welcome-msg",
        data: {
            name: "Harry"
        }
    });
