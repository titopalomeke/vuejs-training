Computed Properties

Vuejs permite introducir expresiones lógicas en las templates eso puede "manchar" el código y complicar su mantenimiento. Las props computed permiten extraer esa lógica facilitando la claridad del código en nuestras vistas

                    
    <div id="parsel-name">
        <p>Nombre: "{{ name }}"</p>
        <p>Nombre en lengua parsel: "{{ parselName }}"</p>
    </div>
                    
                
                    
    vm = new Vue({
        el: '#parsel-name',

        data: {
            name: 'Alberto'
        },

        computed: {
            parselName: function () {
                return this.name.split('').reverse().join('');
            }
        }
    });

Nombre: "{{ name }}"

Nombre en lengua parsel: "{{ parselName }}"
